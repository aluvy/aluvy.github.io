## 1. JS 파일 생성

````javascript
const methods = {
  globalFuncTest(){
    console.log('global function test');
  }
}

export default {
  install(Vue) {
    Vue.config.globalProperties.$globalFuncTest = methods.globalFuncTest;
  }
}
````

## 2. main.js 파일에서 생성한 파일 글로벌 선언하기

````javascript
(main.js)

import globalMethods from "./assets/js/front"

createApp(App)
  .use(globalMethods)
  .mount('#app');
````

## 3. 함수 사용

````javascript
mounted(){
  this.$globalFuncTest();
},
````
